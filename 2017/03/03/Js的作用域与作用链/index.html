二、作用域链：

对于在ss()的找name,作用域链是： ss()->fun()->window

一个容易出错的例子：

分别点击三个按钮，都弹出了Button4
循环的速度很快，当注册事件结束后，i的值为4，当点击按钮时，事件函数即function(){ alert(“Button”+i);}这个匿名函数中没有i,根据作用域链，所以到buttonInit函数中找，此时i的值为4，所以弹出”button4“。

三、延长作用域链

try-catch 语句的 catch 块；
with 语句（用来临时扩展作用域链，将语句中的对象添加到作用域的头部）。

with语句将ob.ab添加到当前作用域链的头部，所以输出的就是：”b”.
with语句结束后，作用域链恢复正常。
