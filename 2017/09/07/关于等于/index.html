关于等于

  • JavaScript
  • 技术

关于等于。。。哈哈哈，好奇怪的名字呀，不过没关系，这次的主角是等于，那是这个”=”,还是这个”==”,亦或是这个”===”呢，下面说
刚刚才发现object里有一个is方法还挺好玩的

说到等于我们会想起=，
=：一个等号是赋值
==：两个等号是判断相等，但是两个等号会对等号两边的值进行隐式转换，比如“” == false //true
===：三个等号是严格相等，但是对于NAN === NAN // false, +0 === -0 // true

然后这个is是严格完全相等，比===等号还要强一点，但是好用的东西总是不能肆无忌惮的用
这个is完全不支持IE和Safari
